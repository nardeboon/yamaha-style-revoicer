# A/B Comparison — Iranian PSR styles (loudness)

Both folders contain the **same 96 styles with the same voices** (natural CVP
upgrades) and the same fixes (rhythm channels forced to real drum kits). They
differ **only in loudness normalization**, because the amateur source styles are
very hot (most channels set to volume 127).

Load the same filename from each and pick the level you prefer:

| | `Iranian-Combined-CVP805-A` | `Iranian-Combined-CVP805-B` |
|---|---|---|
| Channel volume cap (CC7) | **110** (gentle) | **95** (stronger) |
| Note velocity | unchanged | **×0.85** (trimmed) |
| Character | tames only the maxed-out channels, keeps punch | calmer, more even, more headroom |

Tell me **A** or **B** overall (or "between the two" and I'll pick an
intermediate cap).

---

Fixes applied to both (not an A/B choice):
- **Rhythm channels** (ch9/ch10) that were voiced as piano/guitar/strings are now
  forced to real drum kits, so their chord-stab parts play as tuned percussion
  instead of a melodic instrument. A missing bank select is injected where needed.
- **Voices** upgraded from plain GM to CVP-805 panel voices (CFX piano, Classical
  strings, Symphony brass, dynamic guitars, etc.).
